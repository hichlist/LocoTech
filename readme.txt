Для того чтобы запустить проект нужно:
1. Распаковать архив в нужное вам место на диске.
2. Открыть командную строку и перейти в папку с проектом.
3. Далее набрать команду:
	а. Для Linux: python3 manage.py runserver
	b. Для Windows: python manage.py runserver
	* Python должен быть предварительно установлен.
